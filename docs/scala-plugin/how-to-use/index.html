DevOopsScalaPlugin - How to Use

DevOopsScalaPlugin#

This plugin is for a Scala project and is enabled by default. It has quite a few scalacOptions added. If you'd like to see what those options are, simply run

show scalacOptions

e.g.)

[info] * -language:higherKinds
[info] * -encoding
[info] * UTF-8
[info] * -deprecation
[info] * -unchecked
[info] * -feature
